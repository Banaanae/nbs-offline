# NBS Offline

An offline client for v59 that supports Android and iOS.

## Features

- Main menu
- Offline battles
- Config
- Player profile
- Changing name
- Saving thumbnail, name color, favourite brawler and more
- Random bot names
- Content creator codes
- Teams (wip)
- Starr road (wip)
- Tutorial
- Better loading screen
